## The Expense tracker
Built this app using create-react-app(react-js) and and normal css 
 # Features
 - adds new items
 - removes items
 - update Expense and income
 # Want to use it 
 ```
 git clone https://github.com/tresor-01/React-Projects.git
 cd expense-tracker
 npm install 
 npm start

 http://localhost:3000

 ```
